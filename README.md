## Image Grid Layout
A lightweight, pure HTML + CSS tooltip component for navigation items. This project builds a small pop-up “tooltip” box that appears when the user hovers over a navigation item, providing extra context or hints without JavaScript.
- Tooltip appears on hover of nav items
- Absolutely positioned above the trigger element
- Smooth fade-in / fade-out using opacity and transform
- Pure CSS: no JS required
- Customizable styling (colors, fonts, border radius)
- Arrow (triangle) pointer indicating the relation to the nav item
